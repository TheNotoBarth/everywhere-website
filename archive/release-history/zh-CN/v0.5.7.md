**⚠️ 迁移提醒：如果您从 0.5.5 及之前的版本升级，请务必注意。**
- 此版本会自动迁移设置文件，建议在升级前备份 `C:\Users\[Username]\AppData\Roaming\Everywhere\settings.json`
- 迁移将在软件启动后自动完成。所有的旧 API Key 将被清空并移动到 settings.json 的 `LegacyApiKeys` 字段中
- 您需要手动重新配置密钥。为了安全起见，请在重新配置完成后立即删除 `LegacyApiKeys` 字段

- 托盘图标交互优化：现在您可以单击托盘图标打开聊天窗口，双击图标打开主窗口

- **（重要）** 修复了"进阶配置"选项无法正常生效的问题
- 修复了主窗口在最小化状态下无法通过托盘菜单正确显示的问题
