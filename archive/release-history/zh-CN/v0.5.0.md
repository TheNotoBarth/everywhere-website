## ✨ 新特性

- 🎉 新增对 MCP（模型上下文协议）的支持！你现在可以添加自己的 MCP 工具，当助手调用这些工具时会自动启用。支持 Stdio、Streamable HTTP 和 SSE 协议

## 🚀 改进

- 当助手调用的工具未找到时，现进行模糊匹配并告知助手，减少模型幻觉
- (Windows) 为 Everything 插件添加输出显示
- (Windows) 移除 Windows System API 插件
- 其他 UI 调整和错误修复

## 🐛 修复

- 修复了自定义助手头像被拉伸的问题（#155）
- 修复了 Markdown 内容难以选择且复制时可能导致错误的问题（#91）
- 修复了 Markdown 中的粗体或斜体文本在被选中时会恢复为正常字体的问题（#114）
- 修复了捕获 UI 元素无法取消的问题
