## ✨ 新特性

- 插件执行反馈
  - 高权限插件现在需要用户确认才能运行
  - 插件调用的结果现在显示在聊天窗口中
    - 文件系统插件列出访问了哪些文件
    - 文件更改会显示出来，并且可以在应用之前进行审查
    - 网络搜索现在显示正在使用的具体查询
- 临时聊天
  - 不保存的临时聊天现在在切换到另一个聊天或创建新聊天时会自动删除
  - 你可以选择在设置中自动进入临时聊天模式
- 网络搜索增强
  - 添加了 Jina 作为网络搜索服务商
  - 添加了 SearXNG 作为网络搜索服务商
- 添加了控制视觉上下文使用和细节级别的设置
- 聊天窗口现在显示当前聊天标题
- (Windows) 集成 Everything 加速本地文件搜索

## 🚀 改进

- 改进了主窗口的 UI 布局和样式
- 在聊天输入框中启用了右键上下文菜单（复制、剪切、粘贴）
- 在聊天窗口中添加了“滚动到底部”按钮
- 为自定义助手添加了更多 emoji 表情选择

## 🐛 修复

- 修复了聊天历史有时排序不正确的问题
- 修复了 Alt 键无法用作快捷键的问题
- 修复了默认助手的图标无法更改的问题
- 修复了元素选择器无法通过右键关闭的问题
- 修复了选择元素后主窗口会出现的问题
- 修复了聊天过程中可能出现的无响应问题
- 修复了自定义助手的连接测试失败的问题
- 修复了某些模型可能导致聊天标题生成失败的问题
- 修复了字体可能会意外变为斜体的问题
- 修复了欢迎对话框中的死链接问题
- 修复了递归自引用的问题
- 修复了 Windows 10 上错误的亚克力效果问题
