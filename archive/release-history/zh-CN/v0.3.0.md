## ✨ 新特性

- 🎉 新图标
- 新增托盘菜单的亚克力效果
- 为首次使用用户添加了开箱体验（OOBE），包括：
  - 欢迎对话框
  - 快速设置向导
- 新增自定义模型支持
- 新增聊天附件存储功能
- 新增对更多热键的支持，例如 Windows 上的 `Copilot` 键
- 新增守护进程
- 聊天窗口可以手动调整大小
- 当聊天窗口被固定时，会显示在任务栏上

## 🚀 改进

- 重构了插件系统，包括：
  - 在设置中添加了插件管理器
  - 添加了用于读写文件的文件系统插件
  - 添加了在 Windows 上使用 PowerShell 的代码执行插件
  - 添加了使用 Puppeteer 的网页浏览插件
  - 添加了视觉元素插件，用于在 UI 自动化不可用时捕获屏幕内容
  - 重构了网络搜索插件
- 重构了日志系统，采用结构化日志
- 提升了视觉上下文捕获性能
- 提升了亚克力效果的可见度

## 🐛 修复

- 修复了频繁删除或切换聊天记录可能导致崩溃的问题
- 修复了聊天窗口中的表情渲染问题
- 修复了在某些情况下活动聊天窗口可能导致应用程序冻结的问题
- 修复了设置加载/保存的问题
- 修复了切换聊天记录时新建聊天按钮的禁用状态未更新的问题
- 修复了在某些情况下错误地检测到焦点元素的问题
- 修复了选择文本时聊天窗口可能自动滚动的问题

## ⚠️ 已知问题

- 在某些情况下，聊天消息在选择文本时可能会消失
- 在某些情况下，聊天窗口在被固定时可能会闪烁
