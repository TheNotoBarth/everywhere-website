<Callout type="info" title="What if I accidentally close Everywhere?">
You can find the Everywhere icon in the system tray (bottom right corner on Windows, top right corner on macOS) and double-click to open the main interface of Everywhere.
</Callout>

<Steps>
  <Step>
    You can get the latest version of Everywhere <DynamicLink href="/[lang]/download">here</DynamicLink>.
  </Step>

  <Step>
    When you run Everywhere for the first time, you will see the welcome page.

    ![Welcome to Everywhere](/get-started/quick-start/en-US/welcome-page-1.webp)
  </Step>

  <Step>
    Click the **Let's start!** button and select **Login** to access the models through our service.

    ![Choose Model Provider](/get-started/quick-start/en-US/welcome-page-2.webp)
  </Step>

  <Step>
    In the pop-up browser page, choose the appropriate method to log in to your Sylinko Passport.

    ![Log in to Sylinko Passport](/get-started/quick-start/en-US/passport-login.webp)
  </Step>

  <Step>
    After successfully logging in, return to the client and wait for a moment until your account information appears on the page.

    ![Login Successful](/get-started/quick-start/en-US/welcome-page-3.webp)
  </Step>

  <Step>
    Select **Native Mode** and click **Next step**.

    ![Select Model Provider](/get-started/quick-start/en-US/welcome-page-4.webp)

    <Callout type="info">
    In addition to **Native Mode**, you can also manually connect to model providers by selecting **Preset Mode/Advanced Mode**. For more information, please refer to the <DynamicLink href="/[lang]/docs/faq">FAQ</DynamicLink>.
    </Callout>

    <Callout type="warning" title="Payment and Access for Preset/Advanced Modes">
    Most model providers' APIs require paid access. Please ensure you have obtained a valid API Key and have sufficient balance in your account. Additionally, subscriptions and API services from different providers are billed separately, and subscribing to a service does not grant free access to its API.

    You can refer to the <DynamicLink href="/[lang]/docs/model-provider">Model Providers</DynamicLink> section to learn how to obtain API Keys from various model providers.
    </Callout>

    <Callout type="warning">
    Models other than Native Mode do not belong to Everywhere Cloud Service.
    </Callout>
  </Step>

  <Step>
    Select the model you are interested in and click the **Test assistant availability** button at the bottom.

    ![Select Model](/get-started/quick-start/en-US/welcome-page-5.webp)
  </Step>

  <Step>
    If everything goes well, you will see a ribbon, and you can click **Next step**:

    ![Verification Successful](/get-started/quick-start/en-US/welcome-page-6.webp)
  </Step>

  <Step>
    You can modify the default shortcut key (`Ctrl + Shift + E` or `⌘ + Shift + E`) to bring up Everywhere as needed:

    ![Set Shortcut Key](/get-started/quick-start/en-US/welcome-page-7.webp)

    <Callout type="info" title="Copilot Key">
    For Windows users, if your keyboard has a `Copilot` key, you can use it as a shortcut key.
    </Callout>
  </Step>

  <Step>
    You can choose to join the User Experience Program to help us improve Everywhere:

    ![Telemetry Options](/get-started/quick-start/en-US/welcome-page-8.webp)

    <Callout type="warning" title="Privacy">
    Everywhere takes user privacy very seriously. Telemetry data is anonymous and does not contain personally identifiable information, nor does it include sensitive information such as API Keys. You can choose **Do not join** to maximize privacy protection.
    </Callout>
  </Step>
  
  <Step>
    After that, you will enter the main window of Everywhere and can start using it. Note that Everywhere requires additional plugins to support specific features. You can click on the **Chat Plugins** in the sidebar for detailed settings:

    ![Chat Plugins](/get-started/quick-start/en-US/plugin-page.webp)
  </Step>

  <Step>
    At this point, you have completed the initial setup of Everywhere. You can call it out using the shortcut key and start enjoying!

    ![Hello, Everywhere](/get-started/quick-start/en-US/hello-everywhere.webp)
  </Step>
</Steps>
